UQ Swoosh for Blackboard, Edge and UQ Extend

This is a UQ developer portal for URLs to implement into Blackboard, UQ Extend and Edge.

UPDATE: This project is not approved to be implemented into the new rollout in Blackboard Ultra

Why is this not approved for Blackboard Ultra?

A few reasons:
  • This is not a centrally supported tool. Maybe one day I'll be able to get it integrated via a simple LTI, but that's just beyond my skill level at the moment.
  • Although this tool is designed with accessibility in mind, concerns exist that other people who are not familiar with accessibility requirements will not use appropriate WCAG 2.1 accessibility principles when implementing this or other HTML code into Blackboard Ultra.
  • Emphasis has been placed on using HTML block only as a last resort, and instead should be using the Content Market to import in tools, links, images, videos, H5P, Padlets, etc.
