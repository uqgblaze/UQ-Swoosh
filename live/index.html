To use the code:

<iframe title="Lecture Content" src="https://uqgblaze.github.io/UQ-Swoosh/live/lecture.html" height="185px" width="100%" border="0" frameborder="0" scrolling="no" style="overflow:hidden;"></iframe>

  • Change out the name to either lecture, tutorial or additional
  • Minimum height seems to be 165px; max height is about 240px. If the heading isn't loading up, its because your iframe height is too small. Known bug, will deal with it later.
  • Should be auto-adjusting the code if the screen size is less than 768px.
