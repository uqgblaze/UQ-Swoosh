Live Versions

Use these links to iframe in your headings for Blackboard. They are now currently responsive and should automatically adjust based on the viewport size of your iframe - just make sure you set to width to max.

For Lectures: lecture.html

For Tutorials: tutorial.html

For Additional Content: additional.html

For Assessment.... coming soon!

To use the code:

<iframe title="Lecture Content" src="https://uqgblaze.github.io/UQ-Swoosh/live/lecture.html" height="185px" width="100%" border="0" frameborder="0" scrolling="no"></iframe>

  • Change out the name to either lecture, tutorial or additional
  • Minimum height is about 100px; max height is about 240px. If the heading isn't loading up, its because your iframe height is too small. Known bug, will deal with it later.
